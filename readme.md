## CLI

Make sure Rust & cargo are installed

Install trunk

```zsh
cargo install trunk
```

Add the Rust's target to WA (WebAssembly)

```zsh
rustup target add wasm32-unknown-unknown
```

Init Rust front-end

```zsh
cargo new RustFrontEnd
```

Run app locally

```zsh
trunk serve
```
